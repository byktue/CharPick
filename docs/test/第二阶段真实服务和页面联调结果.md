## 3. 后端聊天接口最小验证

执行命令：

```powershell
$body = @{ message = '请用一句话介绍你自己'; temperature = 0.1; top_p = 0.9; max_tokens = 64 } | ConvertTo-Json -Depth 4
Invoke-RestMethod -Method Post -Uri http://127.0.0.1:8000/chat -ContentType 'application/json' -Body $body | ConvertTo-Json -Depth 6 -Compress
```

结果：通过。

说明：

- 接口返回了 `response` 字段。
- 同时返回了 `model` 与 `tokens_used`。
- 说明后端真实聊天链路已可用。
