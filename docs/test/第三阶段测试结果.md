## 1. 已修复的报错

### 1.1 TXT 编码解码错误

测试过程中，后端在处理 `data/test.txt` 时反复报错：

`'utf-8' codec can't decode bytes in position 4091-4092: unexpected end of data`

已修复点：

- 调整了 `backend/source_preprocess/parsers/txt_parser.py` 的编码探测逻辑。
- 对候选编码解码统一采用容错模式，避免在截断字节位置抛出异常。

复测结果：

- `backend.long.advanced_split_novel('data/test.txt', filter_noise=True)` 可正常返回 10 章。
- 后端 `/start-extraction` 任务可以正常进入章节切分和逐章处理阶段。

## 3. 当前仍存在的外部依赖问题

### 3.1 真实提取分发接口受登录态限制

测试真实 `POST /api/v1/extract` 时，后端返回：

- `登录态无效或已过期`

这说明当前浏览器会话中的 token 与后端校验不一致，导致角色库与看板相关的真实分发链路暂时无法继续。

### 3.2 聊天接口受远程模型超时影响

测试 `POST /chat` 时，后端返回：

- `远程模型调用失败: 504 Server Error: Gateway Time-out for url: https://chat.ecnu.edu.cn/open/api/v1/chat/completions`

这说明当前远程学校模型服务存在超时，聊天页可以发送请求，但后端无法获得稳定回复。

### 3.3 角色库页面暂无卡片数据

- `characters` 页面当前能打开，但筛选结果显示暂无符合条件的角色卡。
- 这与真实提取分发接口被登录态限制有关，当前尚未成功生成可在前端展示的 card 记录。
