人设模拟（AI角色/虚拟人/角色扮演）项目主要分三类：**云端平台（直接玩）、开源项目（可部署/二次开发）、工具/API（用于开发）**。下面按类型推荐2026年主流、好用、稳定的项目。

---

## 一、云端平台（开箱即用，适合普通用户）

### 1. Character.AI
- **类型**：AI角色聊天平台
- **特点**：
  - 海量用户自建角色库（名人、虚拟、原创）
  - 对话自然、人设一致性强
  - 支持自己创建角色：写性格、背景、对话风格
- **适合**：纯聊天、角色扮演、人设测试


### 2. 捏Ta（NieTa）
- **类型**：国产AI角色/虚拟偶像创作平台
- **特点**：
  - 中文友好，二次元/写实人设生成
  - 人设+立绘+对话+世界观一体编辑
  - 支持虚拟偶像养成、互动剧情
- **适合**：国漫/小说人设、虚拟主播IP、沉浸式RP


### 3. NovelAI
- **类型**：AI写作+长篇角色扮演
- **特点**：
  - 强“世界书/记忆书”：人设长期稳定不崩
  - 适合长篇故事、DND/跑团式文字冒险
  - 可自定义种族、职业、性格、价值观、禁忌
- **适合**：网文作者、TRPG玩家、长期人设养成


### 4. SillyTavern（前端）
- **类型**：本地RP前端（连接各类LLM）
- **特点**：
  - 高度自由：角色卡、世界书、插件、美化
  - 支持本地模型（Ollama/vLLM）+ 云端API
  - 社区角色卡极丰富（NSFW/SFW都有）
- **适合**：深度RP玩家、本地部署爱好者

## 二、开源项目（可本地部署、开发者向）

### 1. RealChar
- **地址**：GitHub：realchar-ai/RealChar
- **定位**：全栈AI角色平台（Web+移动端+终端）
- **特点**：
  - 一键创建角色：姓名、性格、背景、语音
  - 支持 GPT-4/Claude/本地LLM + 语音TTS/STT
  - 向量数据库记忆，长期人设一致性
- **适合**：快速搭建自己的Character.AI替代品


### 2. OpenAvatarChat（阿里达摩院）
- **地址**：Gitee：Fx_demon/OpenAvatarChat
- **定位**：多模态数字人对话（3D形象+语音+表情）
- **特点**：
  - 单PC可跑：摄像头感知表情+实时语音+3D数字人驱动
  - 支持MiniCPM-o等多模态模型
  - 内置100+数字人形象
- **适合**：虚拟人直播、AI陪伴、数字人项目


### 3. InstantCharacter（腾讯）
- **地址**：GitHub/Hugging Face
- **定位**：单图生成高一致性角色（文生图/姿势/风格）
- **特点**：
  - 一张照片生成同款角色：不同姿势、场景、画风
  - 角色细节稳定：五官、发型、服饰高度一致
  - 兼容Flux.1等主流文生图模型
- **适合**：人设可视化、漫画/游戏角色批量生成


### 4. AIRI
- **地址**：GitHub：mymessageone/AIRI
- **定位**：可感知环境的AI虚拟伴侣（能“看”屏幕/游戏）
- **特点**：
  - 屏幕感知：知道你在玩什么、看什么
  - 深度人设+情感+记忆系统
  - 可和你一起玩游戏、看视频、互动
- **适合**：AI女友/男友、沉浸式陪伴、游戏NPC

### 5. SimsChat
- **地址**：GitHub：Bernard-Yang/SimsChat
- **定位**：学术级人设驱动对话框架
- **特点**：
  - 结构化人设：职业、志向、性格特质、技能、社交圈
  - 带场景约束：角色在不同场合表现不同
  - 强一致性、拒答不符合人设的问题
- **适合**：研究、严谨人设模拟、企业级数字员工

## 三、API/开发工具（用于自己做产品）

### 1. 讯飞星火 - 角色模拟 API
- **特点**：
  - 人设自定义：职业、性格、语言风格、背景
  - 剧情跟随：限定话题、保持场景沉浸
  - 支持微调、WebAPI、按调用计费
- **适合**：APP/游戏NPC、智能客服、虚拟主播

### 2. 字节/豆包 API（角色人设能力）
- **特点**：
  - 强中文人设、情感、语气、风格模仿
  - 支持长上下文、角色记忆、世界设定
- **适合**：国内应用、内容创作、互动小说
